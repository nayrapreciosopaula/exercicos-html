HTML

     HTML e uma linguagem de marcaçao ultilizada para desenvolver sites para a web.
     e composto por elementos definidos por meio de tags,
     e suporta ultilizaçao e scripts externos como por exemplo JavaScript.

CSS

        o CSS e  a sigla para cascade style sheet ou folha de estilos em cascata.
        e ultilizado para estilizar componetes de uma pagina HTML.
        pode ser ultilizado inline dentro da tag, ou fora dela dentro de uma tag style ou ate mesmo em um arquivo externo.
